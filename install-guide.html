INSTALIM NË ANDROID

Dy metoda: Metoda 1 (e rekomanduar) — instalo PWA direkt nga Chrome. Metoda 2 — bëj APK real me Bubblewrap.

METODA 1 — PWA NGA CHROME (E LEHTË, 30 sekonda)

01
Hap Chrome në Android
Shko te URL-ja e aplikacionit (faqja ku ke publikuar CODE MORS).
Shembull: https://code-mors.vercel.app
02
Shfaqet banner-i "Add to Home Screen"
Chrome e njeh automatikisht si PWA dhe shfaq banner-in në fund. Kliko "Install" ose "Add to Home Screen".
03
Konfirmo instalimin
Shfaqet dialog-u i konfirmimit. Kliko "Install". Aplikacioni shtohet në home screen si ikonë.
04
Hap nga Home Screen
CODE MORS hapet si app i plotë — pa adresë URL, pa browser UI. Duket dhe ndjehet si app nativ.
⚠ Nëse Chrome nuk shfaq banner-in automatikisht: Menu (⋮) → "Add to Home screen" → Install.

METODA 2 — APK REAL (Bubblewrap — për distribut si .apk)

01
Instalo Node.js dhe Bubblewrap
Hap terminal-in në PC:
npm install -g @bubblewrap/cli
02
Inicializo projektin APK
Zëvendëso URL-në me adresën reale të faqes suaj:
bubblewrap init --manifest https://code-mors.vercel.app/manifest.json
Bubblewrap lexon manifest.json dhe konfiguron APK-në automatikisht.
03
Build APK
bubblewrap build
Krijohet skedari app-release-signed.apk në folder-in app/.
04
Instalo APK-në në Android
1. Dërgo APK-në në telefon (Bluetooth, USB, Drive)
2. Aktivizo "Unknown Sources": Settings → Security → Unknown Sources ✓
3. Kliko APK-në → Install
05
Publiko në Google Play (opsionale)
Me APK nga Bubblewrap mund ta ngarkosh direkt në Google Play Console si Trusted Web Activity (TWA).

INSTALIM NË iPHONE / iOS

01
Hap Safari (jo Chrome)
Në iOS, vetëm Safari e mbështet "Add to Home Screen" me plotë funksionalitet PWA.
02
Navigo te CODE MORS
Shko te: https://code-mors.vercel.app (ose URL-ja jote)
03
Kliko Share Icon
Kliko ikonën Share (katror me shigjetë lart ↑) në fund të Safari.
04
Add to Home Screen
Scroll poshtë dhe kliko "Add to Home Screen". Emri është "CODE MORS". Kliko Add.
05
Gati!
Ikona CODE MORS shfaqet në Home Screen. Hapet si fullscreen app.
⚠ iOS 16.4+ mbështet PWA me njoftimet push. iOS 15 mbështet bazën. LocalStorage funksionon në të gjitha versionet.

INSTALIM NË DESKTOP / PC / MAC

01
Hap Chrome ose Edge
Navigo te URL-ja e CODE MORS.
02
Kliko ikonën Install
Në address bar shfaqet ikona ⊕ (Install App). Kliko → Install.
03
Hap si dritare e veçantë
CODE MORS hapet si desktop app i plotë pa UI të browser-it.

🚀 PUBLIKO ONLINE (Hosting Falas)

OPSIONI 1 — VERCEL (Rekomanduar, 2 minuta)

01
Krijo llogari falas në Vercel
Shko te vercel.com → Sign Up me GitHub.
02
Shto skedarët në GitHub
Ngarko 4 skedarë në repository-n tënd: index.html, manifest.json, sw.js, icon-512.svg
03
Import nga Vercel
Vercel → New Project → Import Git Repository → Zgjidh Code-mors → Deploy.
URL jote: code-mors.vercel.app

OPSIONI 2 — GITHUB PAGES (Direkt, falas)

01
Shko te repo-ja Code-mors
github.com/1bush/Code-mors
02
Ngarko skedarët e rinj
Kliko "Add file" → Upload files. Ngarko: index.html, manifest.json, sw.js, icon-512.svg, icon-192.svg
03
Aktivizo GitHub Pages
Settings → Pages → Source: Deploy from branch: main → Folder: / (root) → Save.
URL jote: 1bush.github.io/Code-mors
04
Prit 2-3 minuta
GitHub builds dhe publikon automatikisht. Hap URL-në → instalo PWA!

OPSIONI 3 — NETLIFY (Drag & Drop)

01
netlify.com → Sites → Drag folder
Tërhiq folder-in code-mors/ direkt te Netlify. Publikohet menjëherë me URL unike.
✓ Të 3 opsionet janë 100% falas dhe mbështesin HTTPS (i domosdoshëm për PWA dhe Service Worker).
